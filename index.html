Summary: Latitude vs X

The relationship between various weather variables and Latitude was investigated in this analysis. The data was accessed from OpenWeatherMAP API and then HTML and CSS was used to create visualization to study these relationship.

The data further examined that how cloudiness, humidity, wind speed, and temperature changed as we moved away from equator during the month of January.
